# Mobius Brain

System-wide cognition and orchestration layer for the Mobius Organizational Intelligence Operating System.

## Mission

Coordinate Atlas, Memory, Router, Provider Engine, Pulse, MCMS, Sonic, and Command Center so Mobius behaves like one unified organizational intelligence system.

## Scope

- Intelligence Loop orchestration
- Long-term goal awareness
- Cross-service planning
- Background reasoning jobs
- Executive insight generation
- Autonomous improvement recommendations under governance

## Rule

Mobius Brain coordinates the system. It does not replace Atlas, Memory, Router, Provider Engine, Pulse, or MCMS. Each department keeps clear ownership.
